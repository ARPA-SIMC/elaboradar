ElaboRadar
===============================================================



Introduction
------------

Library of functions to manipulate radar volumes and images.
Functions includes:
 * Anomalous propagation removal
 * Beam blockage corrections
 * VPR correction

ARPAE-specific processing tools
------------------------------

These are found in `src/`
